# Product Catalog / Mini Shop

A small React + TypeScript product catalog with cart functionality.

## Features

- Fetch products from API
- Search products
- Filter by category
- Sort products
- Product details modal
- Add products to cart
- Increase/decrease quantity
- Stock limit validation
- Cart drawer
- LocalStorage cart persistence
- Loading and error states
- Retry loading products
- Responsive UI
- Basic accessibility improvements

## Tech Stack

- React
- TypeScript
- Vite
- CSS Modules / CSS files
- DummyJSON API

## How to run

```bash
npm install
npm run dev
